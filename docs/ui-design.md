# MorningPlus UI Design

## 页面结构

Home（首页）
├── Add Alarm（添加闹钟）
├── Alarm Detail（闹钟详情）
├── Alarm Ring（铃声）
├── Record Voice（录音）
├── Daily Quote（每日一句）
├── Pomodoro（番茄钟）
└── Settings（设置）

---

# 1. Home（首页）

显示：

- 当前时间
- 今天日期
- 今日一句
- 闹钟列表
- 添加闹钟

---

# 2. Add Alarm（添加闹钟）

包含：

- 时间
- 重复
- 铃声
- 标题
- Emoji
- 保存

---

# 3. Alarm Detail（闹钟详情）

可修改：

- 时间
- 重复
- 铃声
- 标题
- 删除

---

# 4. Alarm Ring（铃声）

支持：

- 手机音乐
- 我的录音

---

# 5. Record Voice（录音）

支持：

- 开始录音
- 停止录音
- 播放试听
- 保存录音

---

# 6. Daily Quote（每日一句）

支持：

- 心灵鸡汤
- 名人名言
- 自定义语录

---

# 7. Pomodoro（番茄钟）

支持：

- 25 分钟专注
- 自定义时间

---

# 8. Settings（设置）

包含：

- 深色模式
- 相册背景
- 数据恢复
- 关于应用
